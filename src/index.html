Welcome to the Tour API

Your one-stop solution for managing tour-related operations

Use the following endpoints for various operations:

  • GET /tour: Retrieve all tours
  • GET /tour/:id: Retrieve a specific tour
  • POST /tour: Create a new tour
  • PUT /tour/:id: Update a specific tour
  • DELETE /tour/:id: Delete a specific tour

Note : Replace :id by actual Tour ID
